Verificar Conta

Insira o código enviado ao seu e‑mail

  1. 1. Verifique a caixa de Spam/Lixo Eletrônico: Muitos provedores (como Gmail e Outlook) podem enviar mensagens automáticas para lá por engano. Se estiver no Spam, clique em "Não é spam" para receber os próximos avisos na caixa de entrada.
  2. 2. Aguarde de 1 a 2 minutos: O envio é processado instantaneamente, mas alguns servidores de e-mail podem demorar um pouco para atualizar a caixa de entrada.
  3. 3. Verifique o endereço digitado: Confirme se você digitou seu e-mail corretamente durante o cadastro. Se notar algum erro, clique no botão de voltar no topo para fazer o cadastro com o e-mail correto.
  4. 4. Reenvie o código: Use o link de "reenviar o código" logo abaixo para gerar e enviar uma nova chave de verificação.

Não recebeu o e‑mail? Verifique a pasta de Spam ou reenvie o código.
